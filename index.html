WHY THIS PROGRAMME IS FOR YOU

Frontmatters was founded in 2018 by St. Asuzu Alfred P. M.. He believes that the kind of technical sophistication taught by the Frontmatters can benefit at least two million Nigerian youths and probably more.

This programme is driven by his desire to see every young school leaver between ages 15 and 25 who is financially challenged with furthering his/her education, attain self reliance with Web and Software Development.

Frontmatters will nuture you through its well and carefully selected guarnteed courses to the freedom to organize your own life, make your own decisions without needing much help from other people.

Don't wait to be given a fish. Come let's teach you how to fish!

Developer Fundamental / Web Basics

This is where technical sophistication begins. Acquiring the Developer Fundamentals skills, you would have covered three essential tools to becoming a computer geek: the Unix command line, text editors, and version control with Git. When you are done with this level, you will be able to collaborate with other developers around the globe, even if you are not yet a developer. As a technical person, you should know the basics of the WWW (World Wide Web), and this set of courses covers the essentials: HTML, the universal language of the Web; CSS & Layout, which builds an industrial-strength but simple website.

Beginning Development

When you are done with Web Basics and Developer Fundamentals, you would be learning the foundations of developing web applications. We shall start by learning to manipulate the page with JavaScript to do cool stuffs (mind-blowing stuffs). Then, we'll focus on the Ruby language. It is an elegant and powerful language that has taken the Web by storm. We shall start with the Ruby language, and then, will cover Sinatra, a simple Ruby web framework, to get you ready to dive into the Ruby on Rails tutorial.

Professional Web Development

Haven gotten enough foundations laid, you're now ready to learn industrial-strength web development with the full Ruby on Rails Tutorial. The Rails Tutorial teaches you more than enough to be dangerous. By the time you finish this course, you'll be able to build your own professional-grade web applications.
